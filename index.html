1. Little about us

One time, one of our current members felt pretty bored. He created a thread on 4chan's /g/ where he proposed to teach programming (in Python) to those with no programming experience. Around 20 people, some complete newbs, some already accomplished programmers, joined up - #/g/sicp @ Rizon was the result. Since then, the project grew.

We currently:

  • Have projects where all collaborate as a learning experience.
    • Bhottu, our IRC bot, was the result of one of such experiences. Lots of people learned intermediate Python as a result.
  • Work together on tutorials.
    • Our first one was Python, then C++ appeared (still in need of collaborators). Much more are planned.
  • Have a friendly community for aspiring and expert programmers to exchange insights.
  • Occasionally post porn and such.

2. Projects

Since everything we do is project based, you can participate in every area and subject you are intrested in and think you have something to give. Though sometimes it's encouraged to do occasional mundane tasks that may not be so intresting, but would benefit the community. Like this website.. which needs work btw.

Also we are very open for all sorts of ideas, so feel free suggest even silly things. Want to learn a language? You could suggest a project using it.

2.1. Git

We use git as our main tool for collaboration. Git is a distributed revision control system, which allows multiple persons to participate on a software project. It tracks changes in software made by users and allows a lot of control in versions. Even updates and edits to this website is handled trough git.

We have a organization called gentoomen in github, which acts as an umbrella for different projects. You can ask access to the repositories in our irc channel.

There is going to be a short tut on how to use git and links for learning materials here, when somebody writes it.
